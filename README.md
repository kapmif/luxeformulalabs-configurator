# Luxe Formula Labs Nail Polish Configurator

## Overview
Interactive custom nail polish configuration tool for Luxe Formula Labs.

## Deployment on Cloudflare Pages

### Step 1: Create GitHub Repository
1. Go to https://github.com
2. Create new repository: `luxeformulalabs-configurator`
3. Upload all files

### Step 2: Deploy to Cloudflare Pages
1. Login to Cloudflare Dashboard
2. Go to Pages â†’ Create a project
3. Connect to GitHub
4. Select repository: `luxeformulalabs-configurator`
5. Configure build settings:
   - Build command: (leave empty)
   - Build output directory: (leave empty)
   - Root directory: /
6. Click "Save and Deploy"

### Step 3: Configure Custom Domain
1. In Pages project settings
2. Go to "Custom domains"
3. Add custom domain: `config.luxeformulalabs.com`
4. Update DNS settings as instructed
