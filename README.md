# LuxeFormula Labs - Custom Nail Polish Configurator

A professional custom nail polish configuration tool for LuxeFormula Labs.

## Features
- 7-step configuration process
- Real-time 3D preview
- Dynamic pricing calculation
- Global shipping calculator
- Quote request form
- Mobile responsive design

## Deployment
Deployed on Cloudflare Pages at: https://config.luxeformulalabs.com

## Setup
1. Clone the repository
2. Deploy to Cloudflare Pages
3. Configure Formspree for email handling
4. Set up custom domain
